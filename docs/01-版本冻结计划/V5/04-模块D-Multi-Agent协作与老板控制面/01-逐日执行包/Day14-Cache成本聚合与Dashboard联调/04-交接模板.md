4. `GET /projects/{project_id}/cost-dashboard` 核对聚合字段
5. 前端：`apps/web` 执行 `npm.cmd run build`

### 4) fallback / 缺证口径（必须沿用）

- 当 `fallback_contract.heuristic_run_count > 0` 或 `fallback_contract.missing_mode_run_count > 0`：
  - 视为 Day14 fallback 激活（`fallback_active=true`）
  - 不得把当前成本口径写成完整真实 token accounting 闭环
- 当 `fallback_active=false`：
  - 仅代表当前样本均为 `provider_reported`
  - 不代表全项目、全阶段永久无 fallback
