## 3. Day12 直接消费的最小后端合同

### 3.1 sessions

接口：

- `GET /agent-threads/projects/{project_id}/sessions`

最小必需字段：

- `session_id`
- `session_status`
- `review_status`
- `current_phase`
- `context_checkpoint_id`
- `context_rehydrated`
- `latest_intervention_type`
- `latest_note_event_type`
- `summary`
- `started_at`
- `updated_at`

### 3.2 timeline

接口：

- `GET /agent-threads/projects/{project_id}/timeline`

最小必需字段：

- `message_id`
- `session_id`
- `sequence_no`
- `role`
- `message_type`
- `event_type`
- `phase`
- `state_from`
- `state_to`
- `content_summary`
- `content_detail`
- `created_at`

### 3.3 interventions

接口：

- `GET /agent-threads/projects/{project_id}/interventions`

最小必需字段：

- `message_type`
- `event_type`
- `intervention_type`
- `note_event_type`
- `content_summary`
- `content_detail`
- `created_at`

### 3.4 最小状态流

```text
session_started
-> context_recovery_ready
-> execution_started
-> execution_finished
-> (review_passed | review_required -> rework_requested -> boss_note_event)
-> session_finalized
```
