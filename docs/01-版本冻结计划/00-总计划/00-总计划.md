## 5. V4 冻结范围

1. **仓库入口**：支持项目绑定本地主仓库、记录路径安全边界、默认基线分支和工作区摘要。
2. **工作区快照**：支持目录结构、语言分布、忽略目录和变更会话的最小快照。
3. **文件级落点**：支持候选文件定位、代码上下文包，以及任务 / 交付件到变更计划的映射。
4. **变更准备**：支持变更批次、文件重叠提醒、执行前风险预检和人工确认链路。
5. **验证基线**：支持仓库级命令模板、验证运行记录和失败归因扩展。
6. **证据归档**：支持差异摘要、验收证据包、回退重做和仓库复盘收口。
7. **放行闭环**：支持提交草案、放行检查单、一轮 V4 最小闭环演示，以及 Day16 文档收口。
8. **未纳入当前冻结范围**：远程仓库克隆与鉴权、自动 push / 自动创建 PR、多人协作权限、分布式执行器、复杂 AST 索引、向量数据库 / RAG、跨项目仓库编排等。
